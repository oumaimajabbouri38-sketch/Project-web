# 🌐 Fit With You

A modern multi-page fitness website that promotes a healthy lifestyle.  
Developed as part of a web development learning project using HTML, CSS, and JavaScript.

---

## 🌟 Project Overview

Fit With You is designed to encourage users to adopt a healthy lifestyle by providing useful content about fitness, health, running, and nutrition.

---

## 📄 Pages Included

- 🏠 Home Page (modern and clean design)
- 🏋️ Fitness Training
- 💚 Health & Wellness Tips
- 🏃 Running Guide
- 🍎 Nutrition & Diet
- 📩 Contact Page
